# FortiGate IPsec PSK UI

![License](https://img.shields.io/badge/license-MIT-green)

A lightweight Flask-based web frontend for an officially documented FortiGate API workflow.

## ⚠️ Important
This project is a UI wrapper for an official FortiGate API endpoint.

## 🚀 Features
- Web UI
- Tunnel selection
- PSK retrieval
- No credential persistence

## ⚙️ Install

pip install -r requirements.txt
python app.py

Open http://127.0.0.1:5001
